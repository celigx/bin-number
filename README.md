The first six or eight digits of a card number are known as the issuer identification number (IIN) or bank identification number (BIN). These identify the card issuing institution that issued the card to the card holder.

## BIN checker allows you to identify details such as: 
* Issuing institution or bank
* Card brand
* Card type
* Card category
* Country of origin
* Luhn algorithm
